# 快速安装指南

## 方法一：开发模式安装（推荐）

### 1. 打开浏览器扩展管理页面

- **Chrome**: 访问 `chrome://extensions/`
- **Edge**: 访问 `edge://extensions/`

### 2. 启用开发者模式

在页面右上角找到 **"开发者模式"** 开关，将其打开。

### 3. 加载扩展

1. 点击 **"加载已解压的扩展程序"** 按钮
2. 在弹出的对话框中，选择本项目文件夹：
   ```
   C:\Users\Administrator\Documents\Projects\AI 对话抓取小工具
   ```
3. 点击"选择文件夹"确认

### 4. 验证安装

安装成功后，你应该看到：
- 扩展列表中出现 "AI 会话导出工具"
- 扩展图标为 📥 或默认的拼图块图标

## 使用方法

### 基本使用

1. **访问 AI 会话页面**
   - ChatGPT: https://chatgpt.com
   - Claude: https://claude.ai

2. **点击扩展图标**
   - 在浏览器工具栏中找到扩展图标
   - 如果找不到，点击拼图块图标，在弹出的菜单中找到

3. **确认平台检测**
   - 扩展会自动检测当前页面平台
   - 看到 "✓ 已检测：ChatGPT" 或 "✓ 已检测：Claude" 表示成功

4. **选择导出选项**
   - 包含时间戳 ✓
   - 包含模型信息 ✓
   - 包含引用来源 ✓
   - 包含思考过程 ✓

5. **点击"导出当前会话"**
   - 等待导出完成
   - Markdown 文件将自动下载到默认下载目录

### 导出文件示例

导出的 Markdown 文件格式如下：

```markdown
---
标题：Cherry Studio 知识库导入
平台：chatgpt
模型：GPT-4
创建时间：2024-01-15T10:30:00Z
导出时间：2026-04-09T16:30:00Z
消息数量：12
---

## 👤 用户 *(2024-01-15T10:30:00Z)*

我最近正在学习使用 Cherry Studio 的知识库功能。怎么把一整个代码库变成知识库

---

## 🤖 AI *(2024-01-15T10:30:15Z)*

可以，但**不要直接把原始代码仓库整个文件夹无脑丢进去**...

### 引用来源

1. [Cherry Studio 官方文档](https://cherrystudio.com/docs/kb)

```

## 常见问题

### Q: 扩展图标不显示？

A: 点击浏览器工具栏右侧的拼图块图标🧩，在弹出的菜单中找到 "AI 会话导出工具"，点击旁边的图钉图标📌将其固定到工具栏。

### Q: 提示"未识别的平台"？

A: 这表示当前页面不是 ChatGPT 或 Claude。扩展仍然会尝试使用通用方式提取内容，但成功率可能较低。

### Q: 导出的文件是空白的？

A: 可能原因：
1. 页面还没有完全加载 - 刷新页面后重试
2. 会话内容为空 - 确保会话中有实际内容
3. 网络请求拦截失败 - 尝试在设置中关闭"从网络层获取"选项

### Q: 数学公式没有正确导出？

A: 数学公式的导出依赖于页面的渲染方式。如果公式是以 LaTeX 形式存储的，应该能正确导出；如果是图片形式，可能无法转换。

### Q: 思考过程为什么是空的？

A: 只有当 AI 平台在界面上展示了思考/推理摘要时，才能提取到思考过程。如果界面本身没有显示，则无法获取。

## 卸载扩展

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 找到 "AI 会话导出工具"
3. 点击"移除"按钮

## 更新扩展

由于是开发模式安装，更新需要手动操作：

1. 关闭浏览器
2. 替换项目文件夹中的文件
3. 重新打开浏览器
4. 在扩展管理页面点击刷新按钮🔄

## 权限说明

扩展请求的权限：

| 权限 | 用途 |
|------|------|
| `activeTab` | 获取当前标签页信息 |
| `scripting` | 向页面注入脚本 |
| `storage` | 保存用户设置 |
| `downloads` | 下载导出的文件 |

所有数据处理都在本地完成，不会上传到任何服务器。

## 技术支持

如遇到问题：
1. 查看 README.md 了解详细技术说明
2. 检查浏览器控制台日志（F12）
3. 确认扩展已在正确的网站上运行
